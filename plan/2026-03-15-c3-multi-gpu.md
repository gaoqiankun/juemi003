# C3 · 多卡并发
Date: 2026-03-15
Status: done
Commits: none

## Goal
让单机多卡服务器能同时并行处理多个任务，每张 GPU 独立运行一个 worker 进程，API 层任务满时有界拒绝。

## Key Decisions
- 每张 GPU 对应一个独立 worker 进程（CUDA_VISIBLE_DEVICES 隔离），进程启动时加载模型
- GPU_DEVICE_IDS 环境变量（逗号分隔，如 "0,1,2"）控制启用几张卡，默认 "0"（单卡向前兼容）
- Scheduler 改为 GPU slot 感知：维护 N 个 slot，空闲时分发，全忙时排队
- 有界队列：QUEUE_MAX_SIZE 控制最大等待数，超出时 POST /v1/tasks 返回 503
- mock 模式：用 asyncio task 模拟多 worker，无需真实子进程，测试可覆盖
- worker 进程与主进程 IPC：asyncio + multiprocessing（无需外部 broker）
