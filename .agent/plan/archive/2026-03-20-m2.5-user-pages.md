# M2.5 · 用户侧页面
Date / Status: 2026-03-20 / done / Commits: not committed in this session

## Goal
在 Admin Panel（M2）已有的设计系统基础上，实现用户侧 4 个页面：生成主界面、历史记录、3D 预览器、初始化配置。全部复用 tokens.css + primitives.tsx，不引入新依赖。

## Key Decisions

- **不做用户登录**：鉴权用 API Key（存 localStorage），Setup 页面引导首次配置
- **复用设计系统**：tokens.css / primitives.tsx / i18n 已有，直接扩展
- **用户侧 Shell 独立**：不套 admin-shell，做一个更简洁的 user-shell（无侧边栏，顶栏保留主题/语言切换）
- **路由规划**：`/generate`（主入口，`/` 跳此）、`/generations`、`/viewer/:taskId`、`/setup`
- **设计参考**：`stitch-dark-main.zip` + `stitch-light-main.zip`（Stitch 导出）
- **数据**：全部 mock，hook 结构与 admin 系列保持一致

## 页面功能范围

| 页面 | 路由 | 核心内容 |
|------|------|---------|
| Generate | `/generate` | 左：上传区（拖拽/点击）+ 文字 prompt + 参数（质量/格式/seed）；右：实时进度 + 完成后预览缩图 + 跳转 Viewer 按钮 |
| My Generations | `/generations` | 筛选栏（All/Processing/Completed/Failed）+ 卡片网格（缩图/任务名/时间/状态点/下载按钮）+ 空状态设计 |
| 3D Viewer | `/viewer/:taskId` | 居中大视口（Three.js OrbitControls 占位）+ 右侧面板（模型信息/poly/filesize + 下载多格式 .glb/.obj/.usdz）+ 返回按钮 |
| Setup | `/setup` | 居中卡片；API Key 输入（masked）+ Server URL 输入 + Connect 按钮 + 简短说明文字；配置成功跳 `/generate` |
